# Hadley Designs — interactive presentation

This repository holds a **single-file interactive presentation** built for a client conversation: brand, UX, and conversion framing with a live “simulation” slider and supporting sections.

**It is not a commercial website**, marketing site, or production storefront. It is a **pitch / workshop artifact** meant to be opened locally or hosted privately for review—numbers, competitor framing, and projections are directional and labeled in the document for discussion, not as public guarantees.

## What’s inside

- `index.html` — self-contained HTML/CSS/JS (open in a browser; no build step). Netlify and most static hosts use this as the site root at `/`.

## How to view

Open `index.html` locally, or deploy the repo root as a static site (publish directory: **`.`** / project root). `/preview.html` redirects to `/` for old bookmarks.
